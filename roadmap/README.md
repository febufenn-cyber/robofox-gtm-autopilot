# Remaining phases roadmap

`remaining-phases.json` is the machine-readable source of truth for Phases 4–9.

Run:

```bash
python3 scripts/verify_autonomous_build_plan.py
python3 scripts/verify_autonomous_build_plan.py --phase 4 --readiness
```

The command `build` authorizes autonomous repository implementation for the eligible planned phases, but never live credentials, customer contact, CRM mutation, advertising changes, spending, public publishing, production deployment, or constitutional weakening.
