# Learning Optimizer

1. Match predictions to immutable outcomes.
2. Measure calibration and overconfidence.
3. Update beliefs with source and experiment-family traceability.
4. Apply first-party benchmark gates.
5. Score channels across commercial value, economics, founder effort, attribution, and trust.
6. Limit concentration and run sensitivity analysis.
7. Trigger a new Phase 2 decision when evidence materially changes the board.
