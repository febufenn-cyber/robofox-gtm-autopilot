# Operator Console

- Bind localhost unless separately deployed by the founder.
- Verify signed, expiring sessions and server-side permissions.
- Redact secrets and restricted identifiers.
- Show immutable audit and exception history.
- Surface queue leases, dead letters, backup status, and environment identity.
- Keep production deployment and credentials manual.
